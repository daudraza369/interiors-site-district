## üöÄ Quick Upload Steps

### Step 1: Upload Hero Images
1. Go to **Collections ‚Üí Media**
2. Click **"Create New"**
3. Drag & drop or select `hero-interior.jpg`
4. Enter Alt Text: "Modern interior with luxury plantscaping"
5. Click **"Save"**
6. Repeat for `hotel-atrium.jpg` and `restaurant-plants.jpg`

### Step 2: Upload Header Logos
1. Go to **Site Settings ‚Üí Header**
2. Click on **"Brandmark (Default)"** field
3. Select the uploaded `district-brandmark.png` (or upload it first)
4. Repeat for all 5 logo variants

### Step 3: Upload Footer Logo
1. Go to **Site Settings ‚Üí Footer**
2. Click on **"Footer Logo"** field
3. Select `district-logo.png`

### Step 4: Upload Client Logos
1. Go to **Collections ‚Üí Client Logos**
2. For each logo:
   - Click **"Create New"**
   - Upload logo from `logos/` folder
   - Enter Client Name
   - Add Website URL (optional)
   - Save

### Step 5: Assign Hero Images
1. Go to **Pages ‚Üí Home Page ‚Üí Hero Section** tab
2. For each slide:
   - Click on **"Background Image"** field
   - Select the corresponding hero image you uploaded
   - The text content is already pre-filled!

## üí° Pro Tip

You can upload multiple files at once in Payload CMS:
1. Go to **Collections ‚Üí Media**
2. Click **"Create New"**
3. Drag multiple files into the upload area
4. Fill in Alt Text for each
5. Save all at once

This will speed up the process significantly!
